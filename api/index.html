arsh / api

The little JSON status api behind arshnah.in and now.arshnah.in. One source, CORS-open, cached at the edge.

/api/discord-status — presence + activity (lanyard) /api/now-playing — current / last track (last.fm) /api/recent — top artists this week /api/last-commit — latest public push (github) /api/coding — today's coding time (wakatime) /api/contributions — contribution counts (github)
no keys, no secrets, just reads. ← arshnah.in
